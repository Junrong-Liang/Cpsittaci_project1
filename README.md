## Description
This projects contains files associated with academic projects entitled:"'Genomic epidemiolgy and phylodynamics of chlamydia psitaci indicate its origin and global spread".
It contains contents for reproducing some of the analyses of the bioinformatics approach, including, sequentiallly, from quality control of the raw sequencing read data,host genetic information depletion, homology analyses, annotation, phylogentic inference, and genotyping.
